# Library Book Inventory System

## 📖 About the Project

The Library Book Inventory System is a beginner-friendly Python application that helps users manage a collection of books. It allows users to add new books and view all stored books. The application uses file handling to save book records permanently.

## ✨ Features

- Add new books
- View all books
- Store book records using file handling
- Simple menu-driven interface

## 🛠️ Technologies Used

- Python
- File Handling

## 🎯 Learning Outcomes

This project helped me learn:

- Python functions
- Reading and writing files
- Menu-driven programming
- Managing simple records
